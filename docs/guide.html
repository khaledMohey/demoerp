نظام بيع وتوريدات — شرح خطوة بخطوة
الإصدار 1.0 · المخزون بنظام FIFO

بيانات الدخول (المدير):
اسم المستخدم: admin
كلمة المرور: admin123

1. تسجيل الدخول

  1. افتح رابط النظام في المتصفح (كمبيوتر أو موبايل).
  2. أدخل اسم المستخدم وكلمة المرور.
  3. اضغط دخول.
  4. على الموبايل: اضغط أيقونة ☰ أعلى الشاشة لفتح القائمة الجانبية.

2. لوحة التحكم (الرئيسية)

تعرض ملخصاً سريعاً:

  • قيمة المخزون
  • إجمالي المبيعات والتكلفة والربح
  • إجمالي التوريدات
  • أرصدة البنوك والخزنة
  • عدد الموردين والعملاء والشركات
  • الأصناف منخفضة المخزون

3. المخزن / الأصناف

3.1 إضافة صنف جديد

  1. من القائمة: المخزن / الأصناف
  2. اضغط صنف جديد
  3. أدخل: الكود، الاسم، الوحدة، سعر البيع، الحد الأدنى للمخزون
  4. اضغط حفظ الصنف

ملاحظة: الكمية والتكلفة الفعلية تدخل من التوريدات (دفعات FIFO).

3.2 عرض ملخص المخزن

في أعلى الصفحة 4 مربعات:

  • إجمالي تكلفة المخزون — قيمة الدفعات المتبقية بسعر الشراء
  • إجمالي بيع المخزون — قيمة البيع المتوقعة
  • إجمالي الربح — الفرق بين البيع والتكلفة
  • إجمالي الأصناف — عدد الأصناف النشطة

3.3 دفعات FIFO

  1. اضغط دفعات / تعديل على أي صنف
  2. ستظهر كل الدفعات: تاريخ الدخول، المورد، الكمية، المتبقي، سعر الشراء، سعر البيع
  3. الدفعة المعلّمة التالي للخصم هي الأقدم — يُخصم منها أولاً عند البيع

4. الموردين

4.1 إضافة مورد

  1. القائمة → الموردين → مورد جديد
  2. أدخل الاسم والهاتف والبيانات
  3. احفظ

4.2 كشف حساب المورد

  1. من قائمة الموردين اضغط كشف حساب
  2. يعرض: التوريدات الآجلة، المدفوعات، الرصيد المستحق للمورد
  3. للسداد: أدخل المبلغ واختر الحساب (خزنة/بنك) → تسجيل السداد

5. التوريدات (إدخال المخزن)

  1. القائمة → التوريدات → توريد جديد
  2. اختر المورد
  3. اختر طريقة الدفع:
    • كاش — خصم فوري من الخزنة
    • إنستا باي — خصم من حساب إنستا/بنك
    • آجل — يزيد رصيد المورد
    • جزئي — جزء الآن والباقي على المورد
  4. أضف الأصناف: الكمية + سعر الشراء + سعر البيع لكل دفعة
  5. اضغط إدخال للمخزن

كل سطر توريد = دفعة FIFO منفصلة بسعر شرائها.

6. العملاء والشركات

6.1 العملاء (أفراد)

  1. القائمة → العملاء → إضافة عميل
  2. كشف الحساب يظهر المبيعات الآجلة والتحصيلات

6.2 الشركات (B2B)

  1. القائمة → الشركات → إضافة شركة
  2. تُستخدم عند البيع للشركات (آجل / جزئي)
  3. كشف حساب منفصل لكل شركة

7. المبيعات

  1. القائمة → المبيعات → فاتورة جديدة
  2. اختر نوع المشتري: عميل أو شركة
  3. اختر العميل/الشركة
  4. اختر طريقة التحصيل: كاش / إنستا / آجل / جزئي
  5. أضف الأصناف والكميات وأسعار البيع
  6. احفظ الفاتورة

FIFO: النظام يخصم تلقائياً من أقدم دفعة ويحسب التكلفة والربح بدقة.

من تفاصيل الفاتورة ترى من أي دفعة تم الخصم.

8. البنوك والخزنة

  1. القائمة → البنوك والخزنة
  2. عرض أرصدة: الخزنة، البنوك، إنستا باي
  3. تسوية / إيداع / سحب — لتعديل الرصيد يدوياً
  4. آخر الحركات: توريدات مدفوعة، تحصيل مبيعات، سداد موردين
  5. إضافة حساب جديد: خزنة / بنك / إنستا

9. التقارير

  1. القائمة → التقارير
  2. حدد فترة (من — إلى)
  3. تعرض:
    • إجمالي البيع والتكلفة وصافي الربح
    • إجمالي التوريدات
    • قيمة المخزون الحالية
    • ربح الأصناف — لكل صنف: كمية مباعة، إيراد، تكلفة، ربح
    • جدول المخزون الحالي

10. الموظفين والصلاحيات

من الموظفين (للمدير فقط):

11. سير العمل الموصى به

  1. أضف الموردين والعملاء والشركات
  2. أضف الأصناف
  3. سجّل توريداً من المورد (يدخل دفعات FIFO)
  4. أنشئ فاتورة بيع (يخصم FIFO تلقائياً)
  5. راجع التقارير وكشوف الحساب

12. استخدام الموبايل

  • النظام متجاوب مع شاشة الهاتف
  • اضغط ☰ لفتح القائمة، ✕ لإغلاقها
  • الجداول قابلة للتمرير أفقياً
  • يُفضّل Chrome أو Safari حديث
